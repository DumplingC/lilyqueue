# 🎮 遊戲報名排隊系統

簡單、安全的本地報名排隊系統，讓其他玩家透過瀏覽器報名參加你的遊戲。

## 功能特色

- ✅ **即時報名** — 填寫遊戲 ID 即可報名，系統回報排隊順序
- ⏱️ **時間戳記** — 自動記錄報名時間，區分先來後到
- 🎯 **彈性錄取** — 可按時間自動錄取，也可手動勾選
- 📋 **正選＋備取** — 自訂正選與備取名額
- ⚠️ **遲到標記** — 標記遲到者，下次報名自動警示，可設定自動丟備取或不錄取
- 💬 **聊天室** — 報名後可在聊天室即時交流
- 🖼️ **自訂背景** — 上傳背景圖片個性化頁面
- 🔒 **安全措施** — 管理員密碼、速率限制、輸入過濾
- 📊 **匯出名單** — 一鍵複製正選/全部名單

## 快速開始

### 1. 安裝

確保已安裝 [Node.js](https://nodejs.org/)（建議 v18 以上），然後：

```bash
cd 排隊系統
npm install
```

### 2. 啟動系統

```bash
npm start
```

啟動後會顯示：
```
╔══════════════════════════════════════════════════════╗
║        🎮 遊戲報名排隊系統 已啟動！                  ║
╠══════════════════════════════════════════════════════╣
║  管理員面板:  http://localhost:3000/admin             ║
║  報名頁面:    http://192.168.x.x:3000               ║

║  把上面的「報名頁面」網址分享給其他玩家即可！        ║
╚══════════════════════════════════════════════════════╝
```

### 3. 設定管理員密碼

首次進入 `http://localhost:3000/admin` 時會要求設定密碼。

## 其他人怎麼連進來報名？

### 方法一：同一個 Wi-Fi / 區域網路（最簡單）

1. 啟動系統後，終端機會顯示你的區網 IP（例如 `http://192.168.1.100:3000`）
2. 把這個網址分享給其他玩家
3. 他們用手機或電腦的瀏覽器打開即可報名

### 方法二：不同網路 — 使用 ngrok（推薦）

如果玩家不在同一個網路：

1. 安裝 ngrok：到 [ngrok.com](https://ngrok.com/) 免費註冊並下載
2. 啟動系統後，開另一個終端機執行：
   ```bash
   ngrok http 3000
   ```
3. ngrok 會產生一個公開網址（例如 `https://xxxx-xx-xx.ngrok-free.app`）
4. 把這個網址分享給玩家即可

### 方法三：路由器 Port Forwarding

1. 進入路由器設定頁面
2. 設定 Port Forwarding：外部 Port 3000 → 你的電腦內網 IP:3000
3. 把你的公網 IP + Port 分享給玩家

## 管理員操作流程

1. **建立場次** → 設定標題、正選人數、備取人數、遲到處理方式
2. **分享連結** → 把報名頁面網址給玩家
3. **等待報名** → 即時看到新報名通知
4. **錄取方式**：
   - 🔹 **自動錄取**：按報名時間先後，自動分配正選和備取
   - 🔹 **手動錄取**：一個一個點選狀態（正選/備取/未錄取）
5. **公布結果** → 點擊「公布結果」，所有人即時看到結果
6. **關閉場次** → 結束本次報名

## 遲到標記系統

- 在報名列表中，點擊 ⚠️ 按鈕即可標記某人遲到
- 遲到紀錄會**永久保存**在資料庫中
- 下次該玩家報名時，系統會自動顯示「⚠ 此帳號有遲到紀錄」
- 自動錄取時可選擇：
  - 自動移到備取
  - 直接不錄取
  - 不特別處理
- 可在「設定」頁面查看和管理所有遲到紀錄

## 關閉系統

按 `Ctrl+C` 即可安全關閉。資料（遲到紀錄、密碼設定）會保存在 `data/queue.db` 中。下次啟動時會自動載入。
